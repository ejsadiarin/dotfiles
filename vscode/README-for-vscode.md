# Plugins Required:
- Vim (vim emulation)
- multi-command
- settings cycler

# Open settings:
- `Ctrl + Shift + P` -> `Preferences: Open Settings (JSON)`
- `Ctrl + Shift + P` -> `Preferences: Open Keyboard Shortcuts (JSON)`

# Direct copy paste:
- `settings-copy-paste.json`
- `keybindings-copy-paste.json`
